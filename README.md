This project presents an interactive Power BI dashboard designed to analyze sales performance across categories, regions, and time periods. It includes complete data cleaning, transformation, KPI calculations, and visual storytelling to support better business decisions.

# Project Overview
The objective of this project was to turn a raw Kaggle sales dataset into actionable insights through interactive visualizations and advanced DAX calculations. The dashboard helps track Sales, Profit, Profit Margin, Category-wise trends, and Forecasting metrics with drill-down capabilities.

# Data Cleaning & Transformation
Data cleaning and preprocessing were performed using Power Query:
Removed duplicates and handled missing values
Standardized data types (Date, Currency, Text)
Split and merged columns for better analysis
Created custom calculated columns (Profit, Profit Margin)
Transformed raw dataset into a structured analytics-ready format
